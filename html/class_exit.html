Detailed Description

Класс Exit для управления статусом выхода.

Этот класс позволяет управлять статусом выхода, предоставляя методы для получения текущего статуса и изменения его значения.

Метод changeStatusOfExit изменяет статус выхода на противоположный.

Если выход был закрыт (статус 0), после вызова метода выход станет открытым (статус 1) и наоборот.
